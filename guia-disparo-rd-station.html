Guia de Disparo — RD Station Marketing + RD Conversas

Passo a passo para a equipe APM configurar o disparo em massa dos diagnósticos FUNDEB 2026

Parte 1 — E-mail via RD Station Marketing

1 Criar os campos personalizados no RD Station

Antes de importar os contatos, crie estes campos em Configurações → Campos personalizados:

2 Importar o CSV de contatos

Vá em Relacionar → Base de Leads → Importar e suba o arquivo disparo-645-municipios.csv.

Mapeie as colunas do CSV para os campos do RD:

Dica: Para o disparo de secretários, faça uma segunda importação usando EMAIL_SECRETARIO como e-mail e NOME_SECRETARIO como nome, com uma tag/segmentação diferente.
3 Criar o e-mail no RD Station

Vá em Relacionar → Email → Novo Email e escolha "Colar código HTML".

Cole o conteúdo do arquivo:

  • Para prefeitos: email-rd-prefeito.html
  • Para secretários: email-rd-secretario.html

As variáveis já estão no formato do RD Station:

4 Testar e disparar

Antes de disparar para os 645:

  • Envie um teste para seu próprio e-mail (o RD preenche os campos do seu lead de teste)
  • Verifique se o nome do município, valor e link do PDF estão corretos
  • Clique no botão e confirme que o PDF abre corretamente
  • Depois, dispare para a segmentação de prefeitos
Importante — Assunto do e-mail:
Use: [APM] Diagnóstico FUNDEB 2026 — {{ lead.cf_municipio }} pode estar perdendo R$ {{ lead.cf_valor_potencial }}
O assunto personalizado aumenta significativamente a taxa de abertura.

Parte 2 — WhatsApp via RD Conversas (Tallos)

1 Mensagem padrão para disparo

Use a coluna TELEFONE_WHATSAPP do CSV e a coluna LINK_PDF para montar o disparo.

Modelo de mensagem:

Olá, {{NOME}}! Tudo bem? Aqui é a equipe da Associação Paulista de Municípios (APM). Realizamos um diagnóstico FUNDEB 2026 para {{MUNICIPIO}} e identificamos que o município pode estar deixando de captar R$ {{VALOR}} em recursos. ➤ O Censo Escolar é dia 27 de maio — após essa data a janela fecha. 📄 Veja o diagnóstico completo: {{LINK_PDF}} A APM está em parceria com o Instituto i10 oferecendo consultoria especializada para maximizar essa captação. Posso agendar uma conversa rápida para explicar?
2 Mensagem de follow-up (sem resposta)
Olá, {{NOME}}! Aqui é da APM novamente. Enviei o diagnóstico FUNDEB de {{MUNICIPIO}} na semana passada. Já conseguiu dar uma olhada? Faltam poucos dias para o Censo Escolar (27/05). O município tem R$ {{VALOR}} em potencial não captado. 📄 Diagnóstico: {{LINK_PDF}} Posso ligar em 5 minutos para explicar?
Variáveis no Tallos: O RD Conversas (Tallos) utiliza variáveis no formato {{campo}}. Ao importar os contatos do CSV, mapeie as colunas MUNICIPIO, VALOR_POTENCIAL e LINK_PDF para os campos correspondentes no Tallos.

Parte 3 — Dashboard de Acompanhamento

✓ Acompanhe os downloads em tempo real

Após o disparo, acompanhe quais municípios estão baixando os PDFs:

➔ institutoi10.com.br/sistemas/apm/dashboard

O dashboard mostra:

  • Total de downloads
  • Quantos municípios já baixaram (taxa de abertura)
  • Atividade em tempo real (quem baixou, quando)
  • Ranking dos municípios com mais downloads
  • Busca por município específico
Como funciona o tracking: Quando o prefeito/secretário clica em "Baixar Diagnóstico" no e-mail ou no WhatsApp, o link passa por nossa API que registra o download (município, data, hora) e redireciona automaticamente para o PDF. O prefeito não percebe nenhuma diferença — o PDF abre normalmente.
